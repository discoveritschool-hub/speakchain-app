Принципи

ДНК продукту — рішення, на яких усе тримається
Говоріння з першого дня
Людина починає вимовляти фрази в перші хвилини, а не після місяців теорії. Страх говорити прибирається практикою, а не правилами.
Граматика без термінів
Пояснення лише через суть, сигнальні слова й живі приклади. Заборонені слова: tense, auxiliary, predicate. Метод Голіцинського.
Цінність до оплати
Людина заходить прямо в бот через посилання з ID відео — і одразу повторює фразу зі свого ж відео. Результат до оплати, без кроку лендінгу.
Довіра блогера наскрізна
Голос блогера проходить через увесь флоу — від привітання до утримання, а не лише як разове джерело кліку.
Віддавай секрети, продавай реалізацію
За Хормозі: контент безкоштовний, продукт реалізує обіцянку. Воронка показує, що без продукту людина застрягла.
Прогрес видно й він живий
Mastery росте від практики й згасає без неї. Ланцюжок не хочеться розривати — це й тримає щоденну звичку.
Дешевий AI на масштабі
Haiku, ліміти, короткий контекст, rate limiting. Маржа лишається здоровою навіть на Basic.
Студент бачить лише своє
Від студента приховані не лише адмін-функції, а сама видимість ролей dev / admin / блогер. Фінанси бачить тільки super admin.
Методи навчання, на яких побудовано продукт
Shadowing (тіньовий повтор)
Перепрошиває вимову й ритм через повтор живого мовлення → плеєр з loop, запис голосу
Comprehensible input
Засвоєння через зрозумілий контент трохи вище рівня → відео під рівень, субтитри
Output / speaking-first
Навичка через активне говоріння → монологи, дуелі, говоріння з першого дня
AI-фідбек + gap analysis
Миттєво показує помилки й прогалини → наступний урок б'є точково по слабких місцях
Spaced repetition (SRS)
Повтор у момент забування → автокартотека фраз із субтитрів, розумні нагадування
Content-based / персоналізація
Своя лексика, а не з підручника → студент обирає власні відео за темою
Граматика без термінів
Знижує страх граматики → метод Голіцинського: суть + сигнальні слова + приклади
Прогресія CEFR
Карта A1→C2, видно де ти й що далі → 77 тем, XP, mastery, прогрес-бар
Peer practice
Жива розмова з людиною свого рівня → speaking buddy, підбір партнера
Звичка через ланцюжок
Постійність важливіша за інтенсивність → chain, daily check-in, 15 хв/день
Кожен інструмент усередині — це конкретний метод під конкретну мету. Разом вони закривають повний цикл: вхід (input) → практика (output) → зворотний зв'язок (feedback) → закріплення (SRS) → звичка (chain).
